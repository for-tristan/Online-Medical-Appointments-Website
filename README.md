
# Online Medical Appointments

This project is designed to simplify the process of booking, managing, and tracking medical consultations. The system connects patients with doctors seamlessly with scheduling it also reduces waiting time, and improves access to healthcare services.
## Features

- User-Friendly Appointment Booking
- Doctor Dashboard
- Responsive Design
